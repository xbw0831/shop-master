# 基于 web智能视觉应用开发-基于flask购物商城开发
#### 软件架构
工具：Pycharm, XAMPP Control Panel, navicat
需要的软件：Flask, Flask-SQLAlchemy
使用命令行导出
pip freeze > requirements.txt
安装使用
pip install –r requirements.txt
